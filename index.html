Overview

This project focuses on processing seismic data from Mars and the Moon to identify seismic events using advanced AI models. By implementing filtering techniques and machine learning, we aim to improve data transmission efficiency and develop scalable solutions for signal processing.

Installation

To get started with this project, clone the repository and install the required dependencies.

git clone https://github.com/yourusername/seismic-data-processing.git
cd seismic-data-processing
pip install -r requirements.txt

Usage

Follow the instructions below to use the project:

  1. Data Preprocessing: Run the following command to preprocess the raw seismic data provided by NASA.
    2. python scripts/preprocess_data.py
  2. Signal Processing: Apply MATLAB functions for band-pass filtering and wavelet transforms.
  3. AI Model Training: Train the Temporal Convolutional Network (TCN) using Keras and TensorFlow.
    4. python scripts/train_tcn_model.py
